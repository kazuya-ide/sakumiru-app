## アドオン (将来構想)

プランに関係なく追加できるオプションとして:

- **AI黒板生成** (+¥1,000/月) - 写真管理の拡張
- **freee / マネーフォワード連携** (+¥2,000/月)
- **API アクセス** (+¥3,000/月) - 基幹システム連携用

## 機能フラグの実装

機能制限は `src/lib/plans/features.ts` で一元管理:

```typescript
import { hasFeature } from "@/lib/plans/features";

// サーバー側で呼び出し
if (!hasFeature(company.plan, "cost_management")) {
  return Response.json({ error: "プラン制限" }, { status: 403 });
}

// クライアント側: サイドバーで自動的にグレーアウト
```
